Chapter 3 列表

修改、添加和删除元素

  • 在末尾添加元素:list.append()
  • 在列表中插入元素:list.insert(0,'元素')
  • 删除列表中第三个元素:del list[2]
  • 删除列表末尾的元素:list.pop()
  • 弹出列表中任意位置的元素:list.pop(2)
  • 根据值删除元素:list.remove("元素")

组织列表

  • 排序:list.sort(reverse=True) 这是永久的
    • list2 = sorted(list) 这是临时的
  • 反转 list.reverse()
  • 长度 len(list)

列表解析

切片

list[起点:终点:间隔]，包括起点不包括终点

  • 没有指定起点，自动从头开始；没有指定终点，自动到列表末尾；没有间隔，设置为1
  • 可以使用负数来表示起点
  • 复制列表：list_copy = list[:]，否则会造成list_copy是list的引用

元组 turple

元组是不可变数据，使用$()$来定义。例如dimension = (10, 20)

元组不能修改，只能赋值。

if语句

  • 没有 & 或是 |,用的是 and，or和not
  • 检查特定值是否在列表:if specific_val in list
