写在前面

free is a matter of liberty, not price

为什么弃win10呢，因为Windows是硬gbk16编码，在很多的地方会有诸多不便，例如环境变量。而且在一些软件上会非常狗屎，例如anaconda安装上，对安装目录有路径长度要求。

进入了开源世界，用好GitHub的issue和StackOverflow，多去查找提问，贡献力量

血 泪 教 训

  • 一定要养成良好的Linux使用习惯
    • 千万不要随手用su，这是我翻车的原因
    • 用trash替代rm命令（别骂了别骂了）
      • 那么这里就要安装一个trash-cli的软件，这次一定嗷
  • 遇事不决就重启 重启重装重买，送保送修送我

安装

  1. 下载manjaro-kde的iso镜像文件，利用rufus制作启动盘
  2. 对系统进行分区，当然也可以在镜像文件里压缩已有卷。建议还是用win来分区，然后安装的时候覆盖（我没有看不起Linux的意思啊，我只是想让大家开开眼界bushi）
  3. Notice：安装的时候选择语言一定要是中文，不然在wine里中文输入会有无数的坑
  4. 修改好时区等就可以开始安装了

出墙

  1. 安装qv2ray
  2. 使用dark mode，开机启动等
  3. 通过vmess导入
  4. 内核设置有问题的话，注意资源目录在/lib/v2ray下

oh-my-zsh

  1. 修改默认shell为zsh
  2. 安装oh-my-zsh wget https://github.com/robbyrussell/oh-my-zsh/raw/master/tools/install.sh -O - | sh
  3. [optional] 如果执行失败403等，修改/etc/hosts文件，添加151.101.76.133 raw.githubusercontent.com
  4. 修改主题powerlevel10k
    1. git clone --depth=1 https://github.com/romkatv/powerlevel10k.git ${ZSH_CUSTOM:-$HOME/.oh-my-zsh/custom}/themes/powerlevel10k
    2. 在~/.zshrc文件中修改ZSH_THEME="powerlevel10k/powerlevel10k"
    3. 安装字体(不然可能没法显示emoji)，可以装powerline字体，或者Meslo Nerd Font
  5. source ~/.zshrc
  6. follow the instructions

奇怪的软件们

vscode

如果要使用remote-ssh等remote组件的话，记得去安装visual-studio-code-bin，也就是那个commercial license版本的，不然不给你用remote套件

如果之前有用过vscode的话，可以用settings sync插件，同步你的系统配置和插件等

  • python插件的IPython插件Unable to start session for kernel Python 3.8.5 32-bit

chrome
