计算机网络和因特网

Internet

所有连接因特网的设备称为主机(host)或者端系统(end system)，通过通信链路(communication link)和分组交换机(packet switch)连接起来。端系统之间通信时，发送端系统将数据分段并为每段加上首部字节，也就是分组(packet)。分组交换机的两种类型是路由器(router)和链路层交换机(link-layer switch)。端系统通过因特网服务提供商(Internet Service Provider, ISP)接入因特网。端系统、分组交换机等部件都要运行一系列协议(protocol)，最重要的是TCP(Transmission Control Protocol)和IP(Internet Protocol)，TCP/IP。因特网标准由因特网工程任务组(Internet Engineering Task Force, IETF)研发，标准文档成为请求评论(Request For Comment, RFC)。

协议 定义了在两个或多个通信实体之间交换的报文格式和次序，以及报文发送和接收一条报文或其他事件所采取的动作。

Networks

A network is an intricately connected system of objects, devices, or people.

可以通过计算机网络实现各种资源的共享，有提供服务的主机计算。

网络的分类

网络可以抽象为两个部分：

  • Node
  • Link

网络的分类：

  • LAN Local Area Networks 局域网，广播方式通讯
    • 工作范围小，最多几公里，一般由一个单位来维护
    • 广播式的传输，多用户同时使用一个链接 Multi-user access
    • 网络性能好，速度高
    • 出错率低
  • WAN Wide Area Networks 广域网，点对点的方式通讯
    • 工作范围大，不太可能由一个单位

先有广域网，然后有局域网技术。
